# Roadmap

## 2.2: campaign foundation

- Package architecture with compatibility layer.
- Deep paginated Zenodo and Figshare adapters.
- DataCite meta-index adapter.
- Durable campaign state and page checkpoints.
- Pause, resume, stop, heartbeat, and campaign exports.
- Utility classes and token-aware filename classification.

## 2.3: coverage and verification

- Query-partition tree for hard result caps.
- OSF, Dataverse, Dryad, and generic InvenioRDM adapters.
- URL reachability verification without full download.
- Structured event and error reports.
- Campaign-specific report output.

## 2.4: format intelligence

- Extensible format registry.
- MIME and magic-byte inspection.
- AFMReader/SPM-Kit/Gwyddion support matrix.
- Confidence and reasons per format inference.
- Archive inventories with size and compression-ratio analysis.

## 2.5: benchmark manifests

- Human review workflow.
- Raw/processed/code/paper relationship model.
- SPM-Kit validation manifests.
- Versioned tolerance and parameter records.
- Curated public benchmark index.
